    vim paster mode

    1
2
3
4
    # enter paste mode
:set paste
#quit paste mode
:set nopaste

    vim快捷键

    • C-w w 切换分屏
    • C-] 跳转到选择的文件链接
    • C-p CtrlP插件，打开文件浏览列表
    • C-e NEEDtree，打开目录栏

    Vim Buffer

    • :ls # 缓存区列表
    • :bd1 #删除缓冲区第4号文件
    • :b1 #编辑缓存区1号文件
    • :sb2#编辑buff文件2，并分屏

    vim 设置

    • 颜色主题 主题文件在/usr/share/vim/vim74/colors下 修改.vimrc, 例如： 
      1
      colorscheme desert
    • 主题solarized light 
      1
2
3

set background=light
colorscheme solarized

    vim分屏

    • 已经打开文件状态下，再打开另一个文件并分屏 
    1
2
3
    :e filename
:sp   #水平分割
:vsp  #垂直分割
    • 在命令行下直接分屏 
      1
2
      vim -on file1 file2    #垂直分屏
vim -On file1 file2    #水平分屏
    • 窗口大小 
      1
2
      C-w_Shift+`-`  # 最大化当前窗口
C-w _=   #平均分配窗口大小
